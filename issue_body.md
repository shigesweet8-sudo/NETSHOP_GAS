## 背景
NETSHOP GAS の Web化 Phase1 を正式番号順で進めます。

## 正式番号
- [12] プラットフォーム別集計API

## 実装対象ファイル
- `20_api_netshop.gs`

## 目的
統合売上データをもとに、SHOP / プラットフォームごとの件数・売上・利益を集計して返す API を追加する。

## 必須実装
- プラットフォーム別集計を返す関数を追加する
- API 入口を追加する
- SHOP 単位で件数、売上、手数料、送料、仕入、利益を返す
- `doPost` は新規追加しない

## 業務ルール
- 対象シートは `統合売上データ`
- 基準列は F列 SHOP
- 金額系は既存の粗利計算と整合すること
- 既存の月別集計API / ダッシュボードAPI と矛盾しないこと
- SHOP が空の行は除外または無視すること

## 完了条件
- `20_api_netshop.gs` に実コード差分が入ること
- プラットフォーム別集計 API 本体が追加されていること

- 実装ファイル差分が存在すること
