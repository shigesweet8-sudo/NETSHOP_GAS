## 背景
NETSHOP GAS の Web化 Phase1 を正式番号順で進めます。
[8] 一括再計算API は完了したため、次は [11] 月別集計API を実装対象とします。

## 正式番号
- [11] 月別集計API

## 実装対象ファイル
- `20_api_netshop.gs`

## 目的
統合売上データをもとに、月ごとの件数・売上・利益を集計して返す API を追加する。

## 必須実装
- 月別集計を返す関数を追加する
- GAS の既存呼び出し方式に合わせた API 入口を追加する
- `yyyy-MM` 単位で集計結果を返す
- 件数、売上、手数料、送料、仕入、利益を月別に返す
- `doPost` は新規追加しない

## 業務ルール
- 対象シートは `統合売上データ`
- 基準日は D列 DATE
- 集計対象は既存業務ロジックに従う
- 金額系は既存の粗利計算と整合すること
- 既存ダッシュボード系ロジックと矛盾しないこと
- 日付が空の行や不正な日付は安全に除外または無視すること

## 完了条件
- `20_api_netshop.gs` に実コード差分が入ること
- 月別集計 API 本体が追加されていること
- `md/json` だけの変更で終わらないこと
- 実装ファイル差分が存在すること
- 再レビューで、未確認事項だけを理由に `HIGH` / `CRITICAL` と判定されないこと
