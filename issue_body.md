## 実装対象ファイル
- `20_api_netshop.gs`

## 目的
複数の管理IDに対して、粗利再計算を一括実行できる API を追加する。

## 必須実装
- 複数の管理IDを受け取り、粗利再計算を一括実行する関数を追加する
- GAS の既存呼び出し方式に合わせた API 入口を追加する
- 成功件数、失敗件数、更新成功ID一覧、失敗ID一覧を返す
- 存在しない管理IDは失敗として返す
- `doPost` は新規追加しない
