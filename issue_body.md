## 背景
NETSHOP GAS の Web化 Phase1 を進めます。
[18] バリエーション確認API が完了したため、次は [14] 並び替えAPI を実装対象とします。

## 正式番号
- [14] 並び替えAPI

## 実装対象ファイル
- `20_api_netshop.gs`

## 目的
商品一覧を指定条件で並び替えて返せる API を追加する。

## 必須実装
- 並び替え用の関数を追加する
- API 入口を追加する
- 少なくとも以下を扱えるようにする
  - 並び替えキー指定
  - 昇順 / 降順指定
- `doPost` は新規追加しない

## 業務ルール
- 既存の listItems と整合すること
- シート自体の行順を直接変更しないこと
- API レスポンス上の順序だけを変更すること
- 未指定時は既存順を維持すること
- 存在しないキー指定は安全にエラーまたは無視として扱うこと
- 既存 API 基盤を壊さないこと

## 完了条件
- `20_api_netshop.gs` に実コード差分が入ること
- 並び替え API 本体が追加されていること
- `md/json` だけの変更で終わらないこと
- 実装ファイル差分が存在すること

## 非対象
- UI 実装
- [15] キャンセル処理API
- [16] 一括削除 / 無効化API

## 注意
- 調査ではなく実装 Issue
- 1 Issue = 1機能
- 対象外の機能を混ぜないこと
